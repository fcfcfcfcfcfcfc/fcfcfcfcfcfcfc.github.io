Mean Value Theorem

发表于 2022-10-19 更新于 2022-10-26
Proof, corollary and application of the mean value theorem.

Definition of local maximum

Let \(f\) be a real function defined on a metric space \(X\). We say that \(f\) has a local maximum at a point \(p\in X\) if there exists \(\delta >0\) such that \(f(q)\leq f(p)\) for all \(q\in X\) if \(d(p,q)<\delta\).

Lemma 1 (Fermat Theorem)

Let \(f\) be defined on \([a,b]\), if \(f\) has a local maximum at a point \(x\in(a,b)\), and if \(f'(x)\) exists, then \(f'(x)=0\).

Proof

Choose \(\delta >0\) such that \(f(q)\leq f(x)\) if \(q\in[a,b]\) and \(d(q,x)<\delta\), so that \[ a<x-\delta<x<x+\delta<b \] If \(x-\delta<t<x\), then \[ \dfrac{f(t)-f(x)}{t-x}\geq 0 \] Letting \(t\to x\), we see that \(f'(x)\geq 0\).

If \(x<t<x+\delta\), then \[ \dfrac{f(t)-f(x)}{t-x}\leq0 \] Letting \(t\to x\), we see that \(f'(x)\leq 0\).

Hence \(f'(x)=0\).

Theorem 1 (Cauchy Mean Value Theorem)

If \(f\) and \(g\) are continuous real functions on \([a.b]\) which are differentiable on \((a,b)\). Then there exists \(x\in(a,b)\) such that \[ [g(b)-g(a)]f'(x)=[f(b)-f(a)]g'(x) \]

Proof

Define \[ h(t)=[f(b)-f(a)]g(t)-[g(b)-g(a)]f(t) \] Then we need to show \(h'(x)=0\) for some \(x\in(a,b)\).

Since \(h(a)=f(b)g(a)-f(a)g(b)=h(b)\). If \(h\) is constant, the conclusion holds for every \(x\in(a,b)\). If \(h(t)>h(a)\) for some \(t\in(a,b)\). Let \(x\) be a point which attains its maximum. Since \(x\in(a,b)\), Lemma 1 shows that \(h'(x)=0\). If \(h(t)<h(a)\), the theorem holds similarly if we choose \(x\in(a,b)\) which attains its minimum.

Theorem 2 (Lagrange Mean Value Theorem)

If \(f\) is continuous function on \([a,b]\) which is differentiable on \((a,b)\), then there exists a point \(x\in(a,b)\) such that \[ f(b)-f(a)=(b-a)f'(x) \]

Proof
