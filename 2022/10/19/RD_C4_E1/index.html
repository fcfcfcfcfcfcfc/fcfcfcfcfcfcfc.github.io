The problems on continuous

Exercise 1

Suppose \(f\) is a real function defined on \(\mathbb R^1\) which satisfies \[ \lim_{h\to 0}[f(x+h)-f(x-h)]=0 \] for every \(x\in\mathbb{R}^1\). Does this imply that \(f\) is continuous?

Solution

The statement is actually false.

Let \(f:\mathbb R\to\mathbb R\) be defined as \(f(0)=1\) and \(f(x)=0\) for \(x\in\mathbb{R}\backslash\{0\}\). Then for any \(x\in \mathbb{R}\) the condition holds. However, \(f\) is not continuous at \(0\). To prove this, let \(\varepsilon=\dfrac{1}{2}\), then for any \(\delta >0\) and \(0<|x-0|<\delta\) we have \[ |f(x)-f(0)|=1>\dfrac{1}{2} \] This completes the proof.

Exercise 2

If \(f\) is a continuous function of a metric space \(X\) into a metric space \(Y\), prove that \[ f(\overline{E})\sub \overline{f(E)} \] for every set \(E\sub X\). Show, by an example, that \(f(\overline{E})\) can be a proper subset of \(\overline{f(E)}\).

Solution

Since \(x\in E\) implies that \(f(x)\in f(E)\subseteq \overline{f(E)}\). we have \(E\sub f^{-1}(\overline{f(E)})\). Then we have \(f^{-1}(\overline{f(E)})\) is closed because \(\overline{f(E)}\) is closed. Hence \(\overline{E}\subseteq f^{-1}(\overline{f(E)})\). We have \(f(\overline{E})\subseteq\overline{f(E)}\).

Take \(f:\mathbb{R}\backslash\{0\}\to\mathbb{R},x\mapsto\dfrac{1}{x^2}\) and \(E=\mathbb Z\), we have \(f(\overline{E})=\{1,\frac{1}{4},\cdots\}\). and \(\overline{f(E)}=\{0\}\cup\{1,\frac{1}{4},\cdots\}\). Therefore \(f(\overline{E})\) is a proper subset of \(\overline{f(E)}\).

Exercise 3

Let \(f\) be a continuous real function on a metric space \(X\). Let \(Z(f)\) be the set of all \(p\in X\) at which \(f(p)=0\). Prove that \(Z(f)\) is closed.

Solution

We have \(Z(f)=f^{-1}(\{0\})\). Since \(Z(f)\) is closed because \(\{0\}\) is closed.
