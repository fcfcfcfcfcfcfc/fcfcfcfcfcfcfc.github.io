Exercise 3.1.6

向量 \(a=i+2j-2k\) 是单位向量吗? 如果不是, 求与 \(a\) 同方向的单位向量.

Solution

因为 \(a=(1,2,-2)\), 于是 \(\Vert{a}\Vert=3\neq 1\), 即 \(a\) 不是单位向量, 且与 \(a\) 同方向的单位向量为 \(a^\circ=\left(\dfrac{1}{3},\dfrac23,-\dfrac{2}{3}\right)\).

Exercise 3.1.8

已知向量 \(a\) 的三个方向角相等且都为锐角, 求 \(a\) 的方向余弦. 若 \(\|a\|=2\), 求 \(a\) 的坐标.

Solution

由题意可得, 设 \(a\) 的方向角大小为 \(\theta\), 则 \[ 3\cos^2\theta=1 \] 即 \(\cos \theta=\dfrac{\sqrt3}{3}\).

若 \(\|a\|=2\), 则 \(a\) 的坐标为 \(\left(\dfrac{2\sqrt{3}}{3},\dfrac{2\sqrt{3}}{3},\dfrac{2\sqrt{3}}{3}\right)\).

Exercise 3.1.10

已知向量 \(a=(-2,3,x)\) 与向量 \((y,-6,2)\) 共线, 求 \(x,y\) 的值.

Solution

因为 \(a,b\) 共线, 于是 \(\dfrac{-2}{y}=\dfrac{3}{-6}=\dfrac{x}{2}\). 解得 \(x=-1,y=4\).

Exercise 3.1.11

已知三个力 \(F_1=(1,2,3),F_2(-2,3,-4),F_3(3,-4,-1)\) 作用于一点, 求其合力 \(F\) 的大小与方向.

Solution

\(F=F_1+F_2+F_3=(2,1,-2)\). 则 \(\|F\|=3\), 且 \(F\) 的方向角为 \[ \alpha=\arccos \dfrac{2}{3},\beta=\arccos \dfrac{1}{3},\gamma=\arccos\left(\dfrac{-2}{3}\right). \]

Exercise 3.1.12

已知点 \(P(1,2,3)\), \(Q(2,3,4)\), 求向量 \(\mathbf{PQ}\) 的模与方向余弦.

Solution

因为向量 \(\overrightarrow{PQ}=(1,1,1)\). 故 \(\Vert\overrightarrow{PQ}\Vert=\sqrt{3}\), 其方向余弦为 \(\dfrac{\sqrt3}{3},\dfrac{\sqrt3}{3},\dfrac{\sqrt3}{3}\).

Exercise 3.1.16

把向量 \(a=(-7,4,7)\) 表示为不共面向量 \(e_1=(3,2,1)\), \(e_2=(7,5,0)\), \(e_3=(-2,3,4)\) 的分解式.

Solution

设 \(a=k_1e_1+k_2e_2+k_3e_3\), 则解得 \(k_1=-1,k_2=0.k_3=2\). 即 \(a=-e_1+2e_3\).

Exercise 3.1.18

设点 \(A(1,1,1), B(1,2,0)\), 点 \(P\) 把线段 \(AB\) 分成两段的比为 \(2:1\), 求点 \(P\) 的坐标.

Solution

由题意可知 \(\dfrac{AP}{BP}=2\), 则 \(P=\left(1,\dfrac{5}{3},\dfrac{1}{3}\right)\).

Exercise 3.2.2

设向量 \(b\) 与 \(a=2i-j+2k\) 共线, 且 \(a\cdot b=-18\), 求向量 \(b\).

Solution

因为 \(a\) 与 \(b\) 共线, 于是 \(a\cdot b=\|a\|\cdot \|b\|=18\). 解得 \(b=-4i+2j-4k\).

Exercise 3.2.3

已知 \(\|a\|=4\),\(\|b\|=2\),\(\|a-b\|=2\sqrt{7}\), 求 \(a\) 与 \(b\) 的夹角.

Solution

由余弦定理可知 \[ \cos(a,b)=\dfrac{\Vert a^2\Vert|+\Vert b\Vert^2-\|a-b\|^2}{2\|a\|\|b\|}=-\dfrac{1}{2} \] 即 \(a,b\) 之间的夹角为 \(\dfrac{2}{3}\pi\).

Exercise 3.2.7

设 \((a+3b)\perp(7a-5b)\), \((a-4b)\perp(7a-2b)\), 求 \(a\) 与 \(b\) 之间的夹角.

Solution

由题意有 \[ \begin{aligned} 7a^2+16ab-15b^2=0 \ (1)\\ 7a^2-30ab+8b^2=0 \ (2) \end{aligned} \] \((1)+(2)\) 可得 \(\|a\|=\|b\|\), \((1)-(2)\) 可得 \(\cos(a,b)=\dfrac{\|b\|}{2\|a\|}\). 于是有 \(\cos(a,b)=\dfrac{1}{2}\), 即夹角为 \(\dfrac{1}{3}\pi\).

Exercise 3.2.8

设向量 \(a=(1,1,-4),b=(1,-2,2)\). 求 \(a\) 在 \(b\) 上的射影以及 \(a\) 在 \(b\) 上的射影向量.

Solution

\(a,b\) 之间夹角的余弦值 \(\cos\theta=\dfrac{ab}{\|a\|\|b\|}\), 于是 \(a\) 在 \(b\) 上的射影 \((a)_b=\|a\|\cos\theta=\dfrac{ab}{\|b\|}=-3\), 且 \(\mathrm{Proj}_ba=(-1,2,-2)\).

Exercise 3.2.11

设单位向量 \(a,b,c\) 满足 \(a+b+c=0\). 求 \(a\cdot b+b\cdot c+c\cdot a\).

Solution

因为 \((a+b+c)\cdot(a+b+c)=0\), 于是有 \(a^2+b^2+c^2+2P=0\), 其中 \(P=a\cdot b+b\cdot c+c\cdot a\). 解得 \(P=-\dfrac{3}{2}\)

Exercise 3.2.13

若 \(a\times b=c\times d\) 且 \(a\times c=b\times d\), 证明: \(a-d\) 和 \(b-c\) 共线.

Solution

原题即证明 \((a-d)\times(b-c)=0\). 因为 \[ (a-d)\times(b-c)=a\times b+b\times d-a\times c-c\times d \] 而 \(a\times b=c\times d,a\times c=b\times d\). 故 \((a-d)\times (b-c)=0\).

Exercise 3.2.14

设一个四面体 \(\Omega\) 的顶点为 \(A,B,C,D\). 向量 \(u,v,w,r\) 分别垂直于 \(\Omega\) 的四个表面, 且都指向 \(\Omega\) 的外部, 其大小分别等于其对应的表面的面积. 证明: \(u+v+w+r=0\).

Solution

不妨设 \(u\) 对应平面 \(\triangle ABC\), \(v\) 对应 平面 \(\triangle ABD\), \(w\) 对应平面 \(\triangle ACD\), \(r\) 对应平面 \(\triangle BCD\). 则有 \[ u+v+w+r=\dfrac{1}{2}(\overrightarrow{AC}\times\overrightarrow{AD}+\overrightarrow{AB}\times\overrightarrow{AC}+\overrightarrow{AD}\times\overrightarrow{AB}+\overrightarrow{CD}\times\overrightarrow{BD}) \] 根据直积的运算规律及四面体的性质化简得 \(u+v+w+r=0\).

Exercise 3.2.18

求以 \(A(3,0,0),B(0,3,0),C(0,0,2),D(4,5,6)\) 为顶点的四面体的体积.

Solution

设其体积为 \(V\), 则 \[ V=\left|\dfrac{1}{6}[\overrightarrow{AB}\ \ \ \overrightarrow{AC} \ \ \ \overrightarrow{AD}]\right|=15 \]

Exercise 1

求满足下列条件的平面的方程:

  1. 过直线 \(L_1:x-1=\dfrac{y-2}{0}=3-z\), 且与 \(L_2:\dfrac{x+2}{2}=y-1=z\) 平行.
  2. 平行于平面 \(\pi_1:5x-14y+2z+36=0\), 且与此平面的距离为 \(3\).
  3. 过点 \((2,-1,5)\), 与直线 \(L_1:\left\{\begin{aligned}4x+y+2z=3\\5x+2y+3z=2\end{aligned}\right.\) 平行, 且与平面 \(\pi_1:2x-y+z=1\) 垂直.
  4. 过点 \((2,1,3)\) 及直线 \(\dfrac{x}{2}=\dfrac{y+1}{3}=\dfrac{z-3}{2}\).
  5. 过点 \((3,4,-2)\) 且垂直于两平面 \(2x-4y-3z+5=0\) 与 \(3x-3y-4z+11=0\) 的交线.

Solution

本题中均设所求平面为 \(\pi\), 且平面的法向量为 \(\mathbf{n}=(A,B,C)\).

  1. 由题意可得 \(\mathbf{n}\perp \mathbf{a_1},\mathbf{n}\perp \mathbf{a_2}\), 解得 \(A=C=-\dfrac{1}{3}B\). 又 \(L_1\) 在 \(\pi\) 上, 有 \(A+2B+3C+D=0\), 解得 \(D=2A\). 于是有 \(\pi:x-3y+z+2=0\).
  2. 因为 \(\pi\parallel \pi_1\), 于是设 \(\pi:5x-14y+2z+D=0\), 因为 \(P=(0,0,-18)\in\pi_1\), 于是有

\[ d=\dfrac{|-36+D|}{\sqrt{A^2+B^2+C^2}}=\dfrac{|-36+D|}{15}=3 \]

解得 \(D=81\) 或 \(D=-9\).

  1. 因为 \(\pi\) 与 \(L_1\) 平行, 有 \(-A+3B+3C=0\), 又 \(\pi\perp \pi_1\), 于是有 \(\mathbf{n}\perp \mathbf{n}_1\), 解得 \(2A-B+C=0\). 因为点 \((2,-1,5)\) 在平面 \(\pi\) 上, 于是直线 \(L_2:\dfrac{x-2}{-1}=\dfrac{y+1}{3}=\dfrac{z-5}{3}\) 在 \(\pi\) 上, 解得 \(\pi:x+7y+5z-20=0.\)
  2. 因为点 \(A(2,1,3), B=(0,-1,3)\) 在 \(\pi\) 上, 故 \(\mathbf n\perp (2,2,0)\) 且 \(\mathbf n\perp \mathbf{a_1}\). 解得 \(A=-B=2C\).

解得 \(\pi:2x-2y+z-5=0\).

  1. 由题意, 求得其交线为 \(\dfrac{x+13}{7}=\dfrac{y}{-1}=\dfrac{z+7}{6}\). 故令 \(\mathbf{n}=(7,-1,6)\), 解得 \(\pi:7x-y+6z-5=0\).

Exercise 2

求满足下列条件的直线的方程:

  1. 过点 \((2,0,-1)\), 且与直线 \(L_1:\left\{\begin{aligned}2x-3y+z-6=0\\4x-2y+3z+9=0\end{aligned}\right.\) 平行.
  2. 过点 \(P_1:(2,-1,3)\) 且与直线 \(L_1:\dfrac{x+3}{7}=\dfrac{y}{0}=\dfrac{z+6}{2}\) 垂直相交.
  3. 过点 \((-1,2,3)\), 垂直于直线 \(\dfrac{x}{4}=\dfrac{y}{5}=\dfrac{z}{6}\), 且平行于平面 \(7x+8y+9z+10=0\).
  4. 过点 \(P_0(-3,5,9)\) 且与直线 \(L_1:\left\{\begin{aligned}y=3x+5\\z=2x-3\end{aligned}\right.\) 及 \(L_2:\left\{\begin{aligned}y=4x-7\\z=5x+10\end{aligned}\right.\) 都相交.

Solution

在本题中, 均设所求直线为 \(L\), 且其方向向量为 \(\mathbf{a}=(l,m,n)\).

  1. 因为 \(L\parallel L_1\), 于是 \(\mathbf{a}\parallel \mathbf{a_1}\), 又 \(\mathbf{a_1}=\begin{vmatrix}\mathbf{i}&\mathbf{j}&\mathbf{k}\\2&-3&1\\4&-2&3\end{vmatrix}=-7\mathbf{i}-2\mathbf{j}+8\mathbf{k}\). 取 \(\mathbf{a}=(7,2,-8)\), 有 \(L:\dfrac{x-2}{7}=\dfrac{y}{2}=\dfrac{z+1}{-8}\).
  2. 因为 \(L\perp L_1\) 且与 \(L_1\) 相交, 于是有 \(7l+2n=0\). 又 \(L_1\) 过点 \(P_2(-3,0,-6)\), 于是有 \(\begin{bmatrix}\overrightarrow{P_1P_2}&\mathbf{a} & \mathbf{a_1}\end{bmatrix}=0\). 解得 \(\mathbf{n}=(2,1,-7)\), 于是 \(L:\dfrac{x-2}{2}=y+1=\dfrac{z-3}{-7}\).
  3. 由题意可得 \(\mathbf{a}\perp (4,5,6)\) 且 \(\mathbf{a}\perp(7,8,9)\), 解得 \(\mathbf{a}=(1,-2,1)\). 于是解得 \(L:x+1=\dfrac{y-2}{-2}=z-3\).
  4. 设过点 \(P_0\) 以及直线 \(L_i\) 的平面为 \(\pi_i(i=1,2)\), 则 \(\pi_1,\pi_2\) 的交线即为所求, 故 \(L:\left\{\begin{aligned}4x-2y+z+13=0\\32x+7y-12z+169=0\end{aligned}\right.\)

Exercise 3

单项选择题.

  1. 设有直线 \(L:\left\{\begin{aligned}x+3y+2z+1=0\\2x-y+10z+3=0\end{aligned}\right.\) 及平面 \(\pi:4x-2y+z-2=0\), 则直线 \(L\).
  1. 平行于 \(\pi\); (B) 在 \(\pi\) 上; (C) 垂直于 \(\pi\); (D) 与 \(\pi\) 斜交.
  1. 设矩阵 \(\mathbf{M}=\begin{bmatrix}a_1&b_1&c_1\\a_2&b_2&c_2\\a_3&b_3&c_3\end{bmatrix}\) 满秩, 则直线 \(L_1:\dfrac{x-a_3}{a_1-a_2}=\dfrac{y-b_3}{b_1-b_2}=\dfrac{z-c_3}{c_1-c_2}\) 与直线 \(L_2:\dfrac{x-a_1}{a_2-a_3}=\dfrac{y-b_1}{b_2-b_3}=\dfrac{z-c_1}{c_2-c_3}\).
  1. 相交于一点; (B) 重合; (C) 平行但不重合; (D) 异面.

Solution

  1. 因为 \(L\) 的方向向量 \(\mathbf{a}\) 与 \(\pi\) 的法向量 \(\mathbf{n}\) 平行, 于是 \(L\) 与 \(\pi\) 垂直. 故答案为 C.
  2. 设矩阵 \(\mathbf{A}=\begin{bmatrix}a_3-a_1&b_3-b_1&c_3-c_1\\a_1-a_2&b_1-b_2&c_1-c_2\\a_2-a_3&b_2-b_3&c_2-c_3\end{bmatrix}\), 则有 \(\det(\mathbf A)=0\), 又 \(\mathbf{M}\) 是满秩矩阵, 于是 \(\mathbf{a_1}\not\parallel \mathbf{a_2}\), 即 \(L_1\) 与 \(L_2\) 相交于一点. 故答案为 A.

Exercise 4

已知平面 \(\pi:3x-y+2z-5=0\) 与直线 \(L_1:\dfrac{x-7}{5}=y-4=\dfrac{z-5}{4}\) 的交点为 \(M\), 若直线 \(L\) 在 \(\pi\) 上过 \(M\) 点, 且与 \(L_1\) 垂直, 试求 \(L\) 的方程.

Solution

因为 \(L\perp L_1\), 于是有 \(\mathbf{a}\perp \mathbf{a_1}\), 又 \(L\) 在 \(\pi\) 上, 于是 \(\mathbf{a}\perp \mathbf{n}\). 解得 \(\mathbf{a}=(3,1,-4)\). 又 \(M(2,3,1)\). 于是 \(L:\dfrac{x-2}{3}=y-3=\dfrac{z-1}{-4}\).

Exercise 5

求直线 \(L:\dfrac{x-3}{2}=\dfrac{y-4}{3}=\dfrac{z-5}{6}\) 与平面 \(\pi:x+y+z=0\) 的交点与夹角.

Solution

将直线转为参数式方程, 有 \[ L:\left\{\begin{aligned}x=2t+3\\y=3t+4\\z=6t+5\end{aligned}\right. \] 故 \(L\) 与 \(\pi\) 的交点为 \(\left(\dfrac{9}{11},\dfrac{8}{11},-\dfrac{17}{11}\right)\). 且 \(L\) 与 \(\pi\) 之间的夹角 \(\theta\) 满足 \[ \sin\theta=\cos(\mathbf{a},\mathbf{n})=\dfrac{|Al+Bm+Cn|}{\sqrt{A^2+B^2+C^2}\sqrt{l^2+m^2+n^2}}=\dfrac{11\sqrt{3}}{21} \] 于是 \(\theta=\arcsin \dfrac{11\sqrt3}{21}\).

Exercise 6

求通过 \(z\) 轴且与平面 \(\pi_1:2x+y-\sqrt{5}z-7=0\) 的夹角为 \(\dfrac{\pi}{3}\) 的平面方程.

Solution

设所求平面为 \(\pi:Ax+By=0\), 则有 \[ \dfrac{|2A+B|}{\sqrt{10}\sqrt{A^2+B^2}}=\cos\dfrac{\pi}{3}=\dfrac{1}{2} \] 解得 \(A=-3B\) 或 \(A=\dfrac{1}{3}B\). 于是 \(\pi:x+3y=0\) 或 \(\pi:3x-y=0\).

Exercise 7

求点 \((1,2,-1)\) 到平面 \(x-2y-z+1=0\) 的距离.

Solution

由距离公式有 \[ d=\dfrac{|Ax_1+By_1+Cz_1+D|}{\sqrt{A^2+B^2+C^2}}=\dfrac{\sqrt{6}}{6} \]
